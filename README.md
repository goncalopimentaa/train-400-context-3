## üîç Understanding the Model

This project uses a three-context model, meaning it takes three previous words to predict the next one. This setup allows the model to provide more relevant predictions based on context. Here‚Äôs a quick overview of how it works:

- **Input**: The model takes in three words.
- **Processing**: Using the softmax regression method, it calculates the probability of possible next words.
- **Output**: The model predicts the most likely next word based on the provided input.
